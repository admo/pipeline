Class template segment

boost::pipeline::segment

Synopsis

// In header: <boost/pipeline/segment.hpp>

template<typename Parent, typename Output> 
class segment {
public:
  // types
  typedef std::remove_reference< typename std::remove_reference< Parent >::type::value_type >::type input_type;   
  typedef Output                                                                                    value_type;   
  typedef std::function< value_type(const input_type &)>                                            function_type;

  // construct/copy/destruct
  segment(const Parent &, const function_type &);

  // public member functions
  template<typename Function> auto operator|(const Function &);
  template<typename OutputIt> void run(OutputIt);
};

Description

Represents a series of connected operations.

Template arguments:

  • Parent: type of the segment before this segment in the pipline; in case of (foo | bar | baz) baz has a type of segment<segment<segment<?, Out>, Out> Out>

    Parent must have a run<OutIt>(OutIt) method and value_type typedef. The method run feeds instances of value_type into a range pointed by the given iterator.

  • Output: type of emitted entries

segment public construct/copy/destruct

  1. segment(const Parent & parent, const function_type & function);

    Creates a new segment by concatenating function after parent.

    Parent and function_type must be copy constructible

segment public member functions

  1. template<typename Function> auto operator|(const Function & function);

    Creates a new segment by concatenating function after *this

    Function must be copy constructible. The Output template argument of the returned segment is deduced from the return type of function. 

  2. template<typename OutputIt> void run(OutputIt out_it);

    Runs the segment.

    Gets the output of the _parent segment, transforms each entry using _function and feeds them into out_it.
